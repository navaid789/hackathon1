# Diagrams for Physical AI & Humanoid Robotics Textbook

This directory contains all diagrams and figures for the textbook.

## Chapter 1 Diagrams

### physical-ai-conceptual.png
**Description**: Conceptual diagram showing the differences between Digital AI and Physical AI systems
**Content**:
- Left panel: Digital AI system (data in, computation, data out) with perfect information, no physical constraints
- Right panel: Physical AI system (sensors → computation → actuators) with physics constraints, sensor noise, actuator limits
- Arrows showing the flow of information and physical interaction
- Labels highlighting key differences: physics constraints, sensor noise, latency, actuation limits
